# Sprint 132 — Stream Endpoint + Token Methods

**Version:** v3.3 | **Status:** Plan | **Risk:** LOW | **Depends:** Sprint 130

---

## Overview

Create `/EmbyStreams/Stream` proxy endpoint and add token generation/validation methods to `StreamUrlSigner`. This enables:
1. HLS manifest proxying for signed M3U8 playlists (prevent CDN URL exposure)
2. Short-lived token generation for `/EmbyStreams/Resolve` endpoint

### Why This Exists

The current `StreamUrlSigner` has `Sign()` and `Verify()` methods for catalog item URLs, but we need:
1. Stream endpoint for proxying HLS manifests with individual stream URLs
2. Token methods for `/EmbyStreams/Resolve` endpoint returns

## Phase 132B — Token Generation Methods

### FIX-132B-01: Add Token Methods to StreamUrlSigner

**File:** \`Services/StreamUrlSigner.cs\` (modify)

**What:**
Add new token generation/validation methods:

\`\`csharp
/// <summary>
/// Generates short-lived tokens for /EmbyStreams/Resolve endpoint.
/// Tokens expire after 1 hour.
/// </summary>
public static string GenerateResolveToken(
    string quality,
    string imdbId,
    string pluginSecret)
{
    var exp = DateTimeOffset.UtcNow.AddHours(1).ToUnixTimeSeconds();
    var message = \`$\{quality}:\{imdbId}:\{exp}\`;
    var sig = ComputeHmacSimple(message, pluginSecret);
    return \`$\{message}:\{sig}\`;
}

/// <summary>
/// Validates a token generated by GenerateResolveToken.
/// Checks both HMAC signature and timestamp expiration.
/// </summary>
public static bool ValidateStreamToken(
    string token,
    string pluginSecret)
{
    if (string.IsNullOrEmpty(pluginSecret))
        return true; // Allow unsigned if no secret configured

    var parts = token.Split(':');
    if (parts.Length != 3) return false;

    var quality = parts[0];
    var imdbId = parts[1];
    if (!long.TryParse(parts[2], out long exp)) return false;

    // Check timestamp expiration
    if (DateTimeOffset.UtcNow.ToUnixTimeSeconds() > exp)
        return false;

    // Verify signature
    var message = \`$\{quality}:\{imdbId}:\{exp}\`;
    var expectedSig = ComputeHmacSimple(message, pluginSecret);

    return parts[2] == expectedSig;
}
\`\`
