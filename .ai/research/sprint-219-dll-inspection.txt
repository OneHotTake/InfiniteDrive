## ChannelManager (Emby.Server.Implementations)
Found in Emby.Server.Implementations.dll. Full method list:
- NO methods contain "Search" in their name
- `strings` grep for "ISearchableChannel" returns ZERO results
- The interface is DEFINED in the SDK but NEVER USED by the server
- GetChannelItems takes (IChannel, String externalFolderId, ...) — only folder routing, no search
- GetChannelItemsInternal takes InternalItemsQuery (has SearchTerm) but only passes FolderId to the channel

## Conclusion
IChannel search is IMPOSSIBLE. The SDK defines ISearchableChannel but Emby's server never reads it.
