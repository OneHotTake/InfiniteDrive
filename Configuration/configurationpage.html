Content Filtering

InfiniteDrive syncs all content from your AIOStreams manifest and any lists added by users. Content filtering controls what appears on the user Discover page based on MPAA/TV ratings from The Movie Database. Once content is in your Emby library, Emby's own parental rating system applies.

Configure your TMDB API key on the Lists tab under List Provider Keys. It's shared with content filtering for MPAA/TV ratings.
User Type Item Has Rating Block Unrated = OFF Block Unrated = ON
Unrestricted (no limit) Any ✅ Show ✅ Show
Restricted (max PG) G, PG ✅ Show ✅ Show
Restricted (max PG) PG-13, R, TV-MA ❌ Block ❌ Block
Restricted (max PG) Unrated ✅ Show ❌ Block
When enabled, users with parental restrictions will NOT see content without known MPAA/TV ratings. Unrestricted users are never affected.

Pre-Warm Cache

      
Attempt to resolve stream URLs for all catalog items and store them in cache. This is optimistic — success depends on whether upstream debrid providers have the content cached and are not rate-limiting. Items that fail will simply be resolved on-demand when played.
