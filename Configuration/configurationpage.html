Connect Your Streaming Provider

InfiniteDrive bridges your AIOStreams instance to Emby. Paste your manifest URL below and everything else follows.

AIOStreams
REQUIRED

Your AIOStreams manifest URL. This is the only required field. AIOStreams supplies your source, your stream links, and optional metadata resolvers — all in one URL.

Don't have one? Create one with Stremio Tools →
AIOMetadata
OPTIONAL

InfiniteDrive uses Emby's built-in metadata engine — including any plugins you have installed — as its primary source of truth. AIOMetadata is a supplementary fallback for titles that Emby cannot identify: obscure releases, certain anime, and content with non-standard IDs. Most setups don't need this.

Cinemeta
OPTIONAL

Cinemeta is Stremio's public catalog. When enabled, InfiniteDrive uses it as a last-resort fallback to fill in titles that neither Emby nor AIOMetadata can identify. Useful for obscure anime and international content.

System RSS Feeds
OPTIONAL

RSS feeds added here are available to every user on this server. Individual users can add their own private feeds in their account settings. Leave blank if you don't need server-wide RSS sources.

Content from system RSS feeds is visible to all users regardless of parental controls. Only add feeds appropriate for your entire audience.

🔐 Security


    

      
Playback Authentication

      
InfiniteDrive uses a unique API key to authenticate playback requests. The key is automatically embedded in all .strm files.

      

        How it works:
        
    
          
  • Your API key is generated once during initial setup.
    • 
          
  • It's stored in your plugin configuration file (plaintext, same as debrid API keys).
    • 
          
  • The key is only used for local playback — not transmitted over internet.
    • 
          
  • Invalid keys return 401, preventing unauthorized playback.
    • 
        

      

      

      
⚠ Config is stored in plaintext (config/plugins/InfiniteDrive.xml). Restrict filesystem access to Emby config directory to trusted users only.

    


    


    

      
Stream Signing Secret

      
The signing secret secures .strm file URLs. It is auto-generated on first load. Rotating it invalidates all existing .strm files — run a full source sync afterwards to regenerate them.

      

        
        
      

      

        ⚠ PluginSecret is empty or too short — stream URL signing will not work. Save settings to auto-generate one.
      

      

        
        ⚠ Regenerated — save settings, then run a full source sync.
